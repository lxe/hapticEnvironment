\doxysubsection{Detailed Description}
This file contains the main function that runs the haptic, graphics, and message listening loops. 

This file is in the global namespace. It accesses haptic data and graphics data through external structs, and keeps track of all the messaging sockets and Chai3D threads through the \mbox{\hyperlink{struct_control_data}{Control\+Data}} extern, which is accessible in other files.
