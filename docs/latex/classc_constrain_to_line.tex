\doxysubsection{Detailed Description}
Constrains movement of the haptic device to a line between two points. 

Creates a line between two points that has a strong magnetic effect. Can be enabled or disabled. As with other haptic effects, this class inherits from Chai3D\textquotesingle{}s c\+Generic\+E\+Ffect.
