\doxysubsection{Detailed Description}
A class for moving point clouds. 

Creates a cloud of points that move in a given direction with a specified coherence. Essentially, the implementation of Movshon \& Newsome\textquotesingle{}s moving dot graphics. Creates a cloud of moving points, where {\itshape  c\% } of points are moving according to the direction and velocity given, while the remainder of the points move in random directions.

This function is called for the moving object in each iteration of the graphics loop. For moving dots, this function computes the next position of each true moving dot based on the specified direction and velocity, and re-\/renders the dot accordingly. Each random dot is assigned a random velocity and direction to move. If the dot moves out of the field of view, it\textquotesingle{}s position is randomly selected to be within the field of view.
