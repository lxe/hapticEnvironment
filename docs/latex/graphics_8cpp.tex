\doxysubsection{Detailed Description}
Functions for setting up and starting the graphics loop. 

The graphics loop is the main loop of the program. Haptics runs in its own loop and messaging is handled by a separate thread. The functions here are responsible for using G\+L\+FW libraries to initialize and update the display.
