Detailed Description

Instance of a critical stability task paradigm.

This class instantiates a Critical Stability Task (CST) object. The CST involves stabilizing a cursor with the haptic robot. Each instantiation of the CST creates a CST cursor. The CST can be instantiated with haptic feedback only, visual feedback only, or both. The haptic feedback uses a position-dependent force field to render state information to the user. For more information on the CST, see Quick et al. (https://www.physiology.org/doi/full/10.1152/jn.00300.2017)

The CST system is governed by the equation:

\[ \frac{dx}{dt} = \lambda (x(t)-u(t)) \]

where \(x(t)\) is the CST cursor position and \(u(t)\) is the hand position, and \(\lambda\) is the degree of instability of the system.
