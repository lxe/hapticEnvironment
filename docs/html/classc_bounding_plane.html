Detailed Description

Creates bounding planes that constrict movement to a 2D plane near the center of the workspace.

Restricts movement of the haptic device to a vertical 2D plane. The stiffness of the bounds can be specified.

Deprecated:
Do not use this class until issues with colllision detection are resolved
