\doxysubsection{Detailed Description}
Instance of a critical stability task paradigm. 

This class instantiates a Critical Stability Task (C\+ST) object. The C\+ST involves stabilizing a cursor with the haptic robot. Each instantiation of the C\+ST creates a C\+ST cursor. The C\+ST can be instantiated with haptic feedback only, visual feedback only, or both. The haptic feedback uses a position-\/dependent force field to render state information to the user. For more information on the C\+ST, see Quick et al. (\href{https://www.physiology.org/doi/full/10.1152/jn.00300.2017}{\texttt{ https\+://www.\+physiology.\+org/doi/full/10.\+1152/jn.\+00300.\+2017}})

The C\+ST system is governed by the equation\+: \[ \frac{dx}{dt} = \lambda (x(t)-u(t)) \] where $x(t)$ is the C\+ST cursor position and $u(t)$ is the hand position, and $\lambda$ is the degree of instability of the system.
