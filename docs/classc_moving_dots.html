Detailed Description

A class for moving point clouds.

Creates a cloud of points that move in a given direction with a specified coherence. Essentially, the implementation of Movshon & Newsome's moving dot graphics. Creates a cloud of moving points, where c% of points are moving according to the direction and velocity given, while the remainder of the points move in random directions.

Constructor for moving dot object.

Parameters
nNumber of points
cCoherence of points, must be between 0 and 1
dDirection in degrees that dots are moving in
mMagnitude or speed at which the dots move

This function generates a dot at a random position in the field of view and adds a pointer to the dots using Chai3D's cMultiPoint object. For each set of moving dots, c*n points are chosen to be the "movingDots", or dots that move in the direction specified. The remainder of the points (n-(c*n)) are dots that move in random directions.

Update function for this moving object.

Parameters
dtThe amount of time that has passed since this function was last called
toolPosPosition of the haptic robot
toolVelVelocity of the haptic robot

This function is called for the moving object in each iteration of the graphics loop. For moving dots, this function computes the next position of each true moving dot based on the specified direction and velocity, and re-renders the dot accordingly. Each random dot is assigned a random velocity and direction to move. If the dot moves out of the field of view, it's position is randomly selected to be within the field of view.
