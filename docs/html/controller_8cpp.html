Detailed Description

This file contains the main function that runs the haptic, graphics, and message listening loops.

This file is in the global namespace. It accesses haptic data and graphics data through external structs, and keeps track of all the messaging sockets and Chai3D threads through the ControlData extern, which is accessible in other files.

bool allThreadsDown ( void  )

Checks if the haptics and messaging threads have exited yet. The graphics loop is in main, and exits when all other threads are down, so it is not checked here.

◆ close()

void close ( void  )

Ends the program. This method does so by setting the "simulationRunning" boolean to false. When false, other threads will exit. To exit gracefully, this method waits until all threads have returned before stopping the haptic tool and exiting the graphic interface.
